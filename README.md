# Project 4 - Driving-License-Request-System
Created an automated driving license request process with approvals, task management, and request closure in ServiceNow.

Objective: 

To automate the end-to-end process for requesting different types of driving licenses.

Key Features Implemented: 
- Service Catalog Item creation with user-friendly form 
- Variables to select license type: 
- Learner License 
- Permanent License 
- Renewal
- Manager Approval workflow 
- Automated task creation for different teams (IT / HR / Admin) 
-  Task lifecycle management until completion 
- Automatic request closure once all tasks are completed

Workflow: 

User submits request - Manager Approval - Tasks Created (IT / HR / Admin)  - Teams Complete Tasks - All Tasks Closed  - Request Closed 

Outcome: 
- Streamlined manual process into automated workflow
- Improved efficiency and tracking
- Enhanced visibility for requesters

Tools & Technologies: 
- ServiceNow
- Flow Designer
- Service Catalog
- Workflow Automation
